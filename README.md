# Checked App

This Rails app allows a user to manage chores and other tasks as part of a team or solo.

## Installation

Clone this repo.
    $ bundle install
    $ rake db:migrate

## Usage

Type the below line into your terminal and then visit localhost:3000 in your web browser.

    $ rails s
